# Menu Calculator (Java Console Application)

This is a Java console-based calculator program that uses a menu system to perform basic mathematical operations.

## Features
- Menu-driven system for user interaction
- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division of two numbers (with zero-check protection)
- Clean method-based structure for each operation
- Continuous loop until user chooses to exit

## Concepts Used
- Methods
- Scanner (user input)
- Loops (while loop)
- Conditional statements (if / else if)
- Basic arithmetic operations
- Program structure and modular design

## Purpose
This project was created to practice Java fundamentals, especially:
- Method creation and usage
- Menu-based program flow
- Handling user input
- Structuring code into reusable components

## Future Improvements
- Allow multiple numbers instead of just two inputs
- Add error handling for invalid input types
- Store previous calculations (history feature)
- Add array-based calculation mode
